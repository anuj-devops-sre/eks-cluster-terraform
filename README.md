# eks-cluster-terraform

Production-grade AWS EKS cluster setup using Terraform.

## What's included
- EKS Cluster with managed node groups
- VPC CNI networking
- IRSA (IAM Roles for Service Accounts)
- Cluster Autoscaler
- HPA configuration
- NGINX Ingress Controller
- Helm chart integration

## Usage
```bash
terraform init
terraform plan
terraform apply
```
